# analisis-clientes-las-varas

Descripción
  Este proyecto corresponde a un análisis exploratorio de datos (EDA) aplicado a una tienda de frutas y verduras con despacho a domicilio llamada Las Varas.
  
  El objetivo fue analizar el comportamiento de compra de los clientes e identificar variables relevantes para apoyar decisiones comerciales.


Dataset

El dataset fue generado mediante simulación en Python con fines académicos.

Incluye información sobre:

  Edad
  
  Género
  
  Comuna
  
  Canal de compra
  
  Número de visitas
  
  Número de compras
  
  Monto total gastado (CLP)
  
  Productos dañados
  
  Calificación del servicio
  

Análisis Realizado

Se desarrollaron las siguientes etapas:

  Análisis estadístico descriptivo
  
  Visualización de datos (histogramas, boxplots, heatmap, scatterplot)
  
  Análisis de correlación
  
  Modelo de regresión lineal


Principales Resultados

  Existe relación positiva entre número de visitas y monto total gastado.
  
  El modelo de regresión obtuvo un R² aproximado de 0,249.
  
  Por cada visita adicional, el gasto aumenta en promedio alrededor de $13.430 CLP.
  
  Los productos dañados afectan negativamente la calificación del servicio.
